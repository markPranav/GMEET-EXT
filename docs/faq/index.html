This service requires a 2 pronged approach to use: Browser extension & controller app; For now we have developed a extension for chrome and a mobile app that is available on both iOS and android.

Gmeet RC is a full fledged mobile app, where as the barebones app is more of a sdk that can be used to integrate with your own custom applications.

The browser extension was built with simple vannila javascript, controller app was developed with flutter and dart.
