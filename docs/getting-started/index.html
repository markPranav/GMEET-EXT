Getting Started

Installation

Overview

You need to install a RC Hub browser extension and a controller App in order to fully achieve the capability of this application.

GMeet RC Hub

  1. Go to https://
  2. Click on install
  3. Done

You may choose to use GMeet RC or DIY Barebones app, current instructions are for the former.

Controller App (GMeet RC)

  1. Scan the QR Code or go to the respect app store links given below.
    or

  3. Done

Usage

Pairing/Configuration

    On Controller APP

  1. Click on the power button to turn the remote on.

  2. Check out the remote code on the app.

    3. On Browser

  3. Open the extension.

  4. Enter code in the input field.

  5. Check the connection status if it says "Connected", you are good go.

    6. On Controller APP

  6. Now on the app, select the appropiate desired channel.

  7. You have now successfully connected/configured/paired the devices.
  8. Done

Basic Usage

  1. Make sure you are paired with the desired channel.

  2. Clicking on mic toggle button / cam toggle button will now work exactly as intended on the meet instace remotely running.

  3. Done

Features

Attendance

  1. Make sure you are paired with the desired channel.

  2. Click on the attendants icon on the remote.

  3. You can read the list or share it, using on screen dialog.
